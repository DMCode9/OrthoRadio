# OrthoRadio 📻

Το OrthoRadio είναι ένας διαδικτυακός ραδιοφωνικός player αφιερωμένος στην αναπαραγωγή Ορθόδοξων Χριστιανικών ραδιοφωνικών σταθμών. Είναι κατασκευασμένο με σύγχρονες τεχνολογίες web και διαθέτει ένα ρετρό περιβάλλον χρήστη που θυμίζει κλασικό ραδιόφωνο, σχεδιασμένο για να προσφέρει μια ομαλή εμπειρία ακρόασης τόσο σε υπολογιστές όσο και σε κινητές συσκευές.

## Χαρακτηριστικά ✨

- **Ζωντανή Αναπαραγωγή (Live Streaming)**: Άμεση πρόσβαση σε Ορθόδοξα ραδιοφωνικά streams (π.χ. Radio Pantokrator).
- **Ρετρό Σχεδιασμός**: Ένα όμορφο περιβάλλον (UI) σχεδιασμένο να μοιάζει με οθόνη φυσικού ραδιοφώνου.
- **Cross-Platform**: Πλήρως responsive σχεδιασμός, βελτιστοποιημένος για οθόνες κινητών.
- **Progressive Web App (PWA)**: Δυνατότητα εγκατάστασης στην αρχική οθόνη του κινητού σας για εμπειρία παρόμοια με native εφαρμογή.
- **Διαδραστικά Στοιχεία**: Λειτουργίες όπως το ρυθμιστικό έντασης (volume knob) που λειτουργεί άψογα σε κινητά και εύχρηστες κάρτες σταθμών.
- **Δυναμικό UI**: Εφέ αυτόματης κύλισης (auto-scroll) κειμένου για μεγάλα ονόματα σταθμών και πληροφορίες.

## Τεχνολογίες 🛠️

- **Frontend**: HTML5, CSS3, TypeScript
- **Εικονίδια**: [Lucide Icons](https://lucide.dev/)
- **Εργαλείο Ανάπτυξης**: [Vite](https://vitejs.dev/)

## Οδηγίες Εγκατάστασης (PWA) 📱

Το OrthoRadio είναι μια Progressive Web App (PWA), πράγμα που σημαίνει ότι μπορείτε να την εγκαταστήσετε απευθείας στη συσκευή σας χωρίς να χρειάζεται να την κατεβάσετε από κάποιο App Store.

### Σε Κινητά (iOS / Android)
1. Ανοίξτε τον browser (Safari στο iOS, Chrome στο Android) και πλοηγηθείτε στη σελίδα της εφαρμογής.
2. Πατήστε το εικονίδιο κοινοποίησης (Share) στο Safari ή το μενού με τις τρεις τελείες (⋮) στο Chrome.
3. Επιλέξτε "Προσθήκη στην οθόνη έναρξης" (Add to Home Screen).
4. Η εφαρμογή θα προστεθεί στην οθόνη του κινητού σας και θα λειτουργεί σαν κανονική εφαρμογή!

### Σε Υπολογιστή (Desktop)
1. Ανοίξτε την εφαρμογή σε έναν υποστηριζόμενο browser (π.χ. Google Chrome ή Microsoft Edge).
2. Στη γραμμή διευθύνσεων (URL bar), αναζητήστε το εικονίδιο εγκατάστασης (συνήθως εμφανίζεται στα δεξιά).
3. Πατήστε το και επιλέξτε "Εγκατάσταση" (Install).
